## 패널이 기각한 것 (그리고 이유)

| 기각된 룰 | 이유 |
|---|---|
| **ADX(14) ≥ 20 하드 게이트** | 기존 4H EMA 갭 0.5% 필터와 중복. 초기 추세(ADX 15~19)를 차단해 기회 손실. 3/3 만장일치 기각. 단, `indicators.adx()`는 테스트와 함께 유틸로 보존 — 향후 사용 가능. |
| **RSI 컨플루언스 게이트** | 구조 필터(눌림+반전봉+돌파)가 이미 모멘텀을 내포. RSI 존은 약한 엣지 대비 복잡도 과다. `indicators.rsi()`는 유틸로 보존. |
| **드로다운 사이징 기본 적용** | $28 계좌에서 리스크가 $0.14까지 줄면 거래소 최소 수량 미달 → 봇 거래 불능. **`RISK_SCALE_ENABLED=true`로 옵트인** (잔고 $200+ 권장). |
| **6시간/본전 시간손절** | 본전 부근 노이즈에 멀쩡한 트레이드를 긁어버림 → 8h + -0.5%로 완화. |
| **1.5% 추격 금지 캡** | 눌림→반전→돌파 정상 리듬이 1.0~1.3% 이탈에서 발생 → 1.5%는 정상 신호 15~20% 차단 → 2.0%로 조정. |
| **24h 거래량 유동성 플로어** | 스캔 대상이 BTC/ETH/SOL/XRP/BNB 메이저뿐이라 무의미한 체크. |

## 진입 게이트 순서 (`strategy.analyze`)

```
캔들 수 확인 (15M ≥210, 4H ≥210)
→ 4H EMA50/200 갭 ≥ 0.5%        (횡보 거부)
→ 최근 3캔들 이동 ≤ 4%           (추격 금지 1)
→ ATR ≤ 가격의 3%                (극단 변동성 거부)      [v3]
→ 4H 추세 방향 + 가격 위치        (레짐)
→ 가격이 4H 두 EMA 사이 ✕        (노이즈 구간 거부)
→ 15M EMA50 눌림 존재            (셋업)
→ 반전봉 (방향 일치 + EMA50 통과)  (트리거)
→ 거래량 ≥ 0.8×SMA20             (유동성 확인)          [v3]
→ EMA50 이탈 ≤ 2%                (추격 금지 2)          [v3]
→ 20봉 스윙 돌파                  (확인)
→ SL = 구조적 레벨, ATR 플로어, ≤5% (리스크 한정)        [v3 강화]
```

진입 후: 펀딩비 게이트(main) → 사이징(risk: 명목 ≥$2, 최소수량, 레버리지 캡) → 일일 한도(guard).

## 파라미터 (config.py)

| 파라미터 | 기본값 | 의미 |
|---|---|---|
| `atr_sl_floor_mult` | 0.8 | SL 최소 거리 = 0.8×ATR(14) |
